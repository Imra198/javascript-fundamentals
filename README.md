# javascript-fundamentals
A beginner-friendly JavaScript project demonstrating core fundamentals such as variables, functions, conditionals, loops, arrays, and console output.

## Technologies Used
- HTML5
- JavaScript (Vanilla)

## Concepts Covered
- Variable declarations (`var`, `let`, `const`)
- Functions (named and anonymous)
- Conditional statements (`if/else`, `switch`)
- Loops (`for`)
- Arrays with mixed data types
- Built-in JavaScript functions
- Console output (`console.log`)
## How It Works
- The HTML file loads the JavaScript file using the `defer` attribute
- All output is logged to the browser console
- Functions and control structures demonstrate basic JavaScript behavior

## Purpose
This project was created as a learning exercise to practice and understand JavaScript fundamentals.

## How to Run
1. Open the `index.html` file in any modern web browser
2. Open the browser developer console to view the output
